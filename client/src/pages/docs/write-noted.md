# Types of blocks

Noted employs block styling within its editor, empowering users to effortlessly create various text formats and styles.

- Paragraph
- Heading
- Checklist
- List
- Quote
- CodeFlask (codeblock)
- Delimiter
- Table

&nbsp;

## CodeFlask (codeblock)

Suported languages are:

- atom
- bash
- clike
- css
- html
- iecst
- javascript
- json
- markdown
- mathml
- plain
- python
- rss
- shell
- ssml
- svg
